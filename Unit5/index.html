Section 1 of 5

Display Settings

Settings → System → Display. The most frequently visited settings page in any IT support role.

Key settings

Resolution — Number of pixels. Use the Recommended value for sharpest image on that display.

Scale — Percentage multiplier for text and icons. Increase this (not decrease resolution) when things look too small on high-DPI displays. Can be set independently per monitor.

Multi-monitor modes — Extend (one large workspace), Duplicate (same image on both), Show only on one. Win + P cycles through modes without opening Settings.

Night light — Reduces blue light on a schedule. Configure in Display → Night light settings.

Section 2 of 5

Power and Sleep

Settings → System → Power. Screen timeout and sleep timeout are independent settings.

Power states

Screen timeout — Screen turns off. Machine still running. Fast to wake.

Sleep — Session preserved in RAM. Very low power draw. Fast to resume (under 5 seconds).

Hibernate — Session saved to disk. Zero power draw. Slower to resume. Not enabled by default on all machines.

Power mode slider — From Best power efficiency to Best performance. Adjusts CPU behavior accordingly.

Section 3 of 5

Installing and Managing Apps

Three paths, three different trade-offs for security, updates, and automation.

The three installation paths

Microsoft Store — Sandboxed MSIX packages. Automatic updates. Safer for shared machines. Not every app is available here.

Traditional installer (.exe / .msi) — Full system access when approved via UAC. Required for most enterprise and legacy software. Manual process each time.

winget — Command-line package manager built into Windows 11. Scriptable and repeatable. Best for IT provisioning and automation. winget install AppName

Section 4 of 5

Default Apps and Printers

Where to go for file associations — and what Device Manager's yellow triangle means.

Default apps

Settings → Apps → Default apps. Search by extension (.pdf, .html) or by protocol (http, https). Windows 11 requires per-extension changes — no single-click category override.

Device Manager

Right-click Start → Device Manager. Yellow triangle = driver problem. Roll Back Driver reverts a bad update. Events tab shows troubleshooting history.

Section 5 of 5

Notifications and Region

Managing interruptions and the time zone setting that breaks authentication.

Do Not Disturb automatic rules

When duplicating display — Activates DND when connected to a projector. Best for presentations.

During specific hours — Schedule DND for evenings, nights, or meetings.

When playing a game — Suppresses alerts during full-screen gaming.

Notifications are held silently — not deleted. They appear in Notification Center when DND ends.

Time zone — why it matters

Kerberos authentication (Active Directory domains) allows a maximum 5-minute clock difference. Entra ID has similar requirements. Wrong time zone = authentication failures on managed machines. Always verify on newly provisioned hardware.
